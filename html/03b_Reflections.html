Reflections

Reflection is where every point on a given image is “mirrored” across a given line. This means that every point on an image is the same distance from a line one way as it is on the other.

When we complete this on the coordinate plane, one way we can do this is by using the x- and y-axis as our line of reflection, meaning that the x- and y-axis are our mirror.

When we reflect a figure over the x-axis, we can write that as (x, y) → (x, y).

When we reflect a figure over the y-axis, we can write it as (x, y) → (−x, y).
