Distance between Two Points

We can find the distance between two points by either using the Pythagorean theorem or the distance formula.

Using the Pythagorean theorem:

  • Find the change in the y values and x values.
  • These would be the lengths of the legs of a right triangle.
  • Substitute the values in a2 + b2 = c2 for a and b.
  • Solve for c.
