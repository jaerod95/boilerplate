Similarity of Polygons

While we can show that figures are similar using transformations, it is not the only method. We can actually determine if a figure is similar using proportions.

A figure is similar if all the sides are proportional to each other and all the angles are congruent. This is very similar to the scale factor that we used in the last lesson.

In order to determine if two figures are similar, you need to set a proportion with the sides of one figure on the top of the fraction and the corresponding sides of the other figure on the bottom of the fraction. Then determine if the ratios are equal; if they are, then the two figures are similar.
