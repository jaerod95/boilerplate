Analyze Data Distributions

When we look at different sets of data, we notice that they gather together in different ways. We can analyze the data based on how it is gathered. The arrangement of data on a number line is called a distribution.

Distribution terms:

Cluster: When all of the information is grouped together, it is called a cluster.

Peak: When you have the most points of data at one value, you have a peak.

Gap: When you see a cluster of data around one set of values, and then a cluster of data around another set of values with a space in between, the space is called a gap.

Outlier: When you have a few or even one data point that is far away from all the other values, it is called an outlier.

Symmetric: When the the distribution of data is the same on the left and the right, it is called symmetric. If there is an outlier in the data, then the data is not symmetric.

If we remove the outlier, then the distribution is roughly symmetric; however, with the outlier in place, the distribution is not symmetric.

Why does it matter whether the data is symmetric? Different measures have to be used to analyze data that is not symmetric. For now, you will not need to know which measures to use; just remember to analyze whether the distribution is symmetric or non-symmetric.
