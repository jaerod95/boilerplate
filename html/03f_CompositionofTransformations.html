Composition of Transformations

Now that we have examined how to do various transformations, we are going to figure out their composition, or which transformations have been used to create the image. We often use these techniques in order to show congruence. If we can move one figure onto another and show them to have the same angles and same length of sides, then the figures are congruent.

One of the most common types of composition of transformations is called the glide reflection. This happens when a figure is reflected and then translated. One example of this would be footprints across the sand, with each footprint being a mirror of each other and then moved forward.

In figuring out which transformations have been used, there are a few things that we can use to help us.

  • See if the figure has been moved up and down and side to side without changing its orientation. This would indicate some sort of translation.
  • See if the figure has a mirror image. This would generally indicate some sort of reflection.
  • See if the figure has been turned. This would indicate some sort of rotation.
  • See if the figure has gotten bigger or smaller. This would indicate some sort of dilation.
