Rotations

In a rotation, we turn a figure about a fixed point. This fixed point is called the center of rotation. When we turn this figure, we do so in a circular direction.

Think of the hands of a clock. When we go in the same direction as a clock goes we call the direction clockwise. When we go in the opposite direction that a clock goes, we call it counter-clockwise. When we turn in a circle, the distance we go around the circle is measured in degrees. If we were to go all the way around the circle, it would be 360 degrees.

The fixed point that we use the most often in our rotation, especially on the coordinate plane, is the origin. The distance from every point in our figure to the origin always remains the same.

This is a 90-degree clockwise rotation.

The 90 degrees clockwise about the origin can be written algebraically as (x, y) → (y, −x).

This is a 180-degree clockwise rotation.

The 180 degrees clockwise about the origin can be written algebraically as (x, y) → (x, −y).

This is a 270-degree clockwise rotation.

The 270 degrees clockwise about the origin can be written algebraically as (x, y) → (−y, x).

Example 1

Triangle ABC has vertices of A(3, 2), B(5, 2), and C(5, 5). Rotate the triangle 180 degrees about the origin.

Example 2

Quadrilateral MNOP has vertices of M(2, 3), N(6, 3), O(6, 5), and P(2, 5). Rotate the quadrilateral 90 degrees clockwise about the origin and give the resulting coordinates.

MNOP′ has vertices of M′(3, −2), N′(3, −6), O′(5, −6), and P′(5, −2).
