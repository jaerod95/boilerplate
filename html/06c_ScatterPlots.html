Scatter Plots

A second way to look at two different types of data is to put the information into a scatter plot, which looks at each individual and compares how they answered two different types of questions. This is used to see if one idea relates to another idea.

For example, if you are trying to determine if the amount of studying relates to the score on a test, you would start by asking each person two questions:

  • How much studying did you do for the test (in hours)?
  • What was your score on the test (as a percent)?

Then you would create a table showing those values:

Amount of Studying (hrs) Score (%)
2 78
10 100
6 85
7 89
9 93

Table 6.8: Table with Values

Once you have these data values, you can create the scatter plot. First you would create a number line across the bottom showing the amount of studying that a person did. This is the x-axis of your scatter plot. Then you would create a second number line going up the side showing their score on the test. This is the y-axis of your scatter plot.

Remember that you need to set up the number lines to include all the data points. You do not have to start at zero on either the x- or the y-axis, but it helps to set up the equation that you use. Next put a point or dot on every point in your chart in the corresponding place on the graph.

If one idea relates to another idea, there is an association or correlation. From this scatter plot, we see that as the amount of studying goes up, the score also goes up. This is called a positive association or positive correlation. Since we see that the points form a straight line, or close to a straight line, we note that it is also a linear association.

Some of the associations we will see:

Positive Association: When the values on the x-axis go up, the values on the y-axis go up as well.
Negative Association: When the values on the x-axis go up, the values on the y-axis go down.
No Association: When there is no obvious pattern between the x and the y values, there is no association.

Example 1

The following data was gathered by a student throughout the springtime with respect to the temperature outside and the amount of snow on the ground.

Temperature (degrees Celsius) Depth of snow (inches)
1 15
3 14.5
3 14
7 12
6 11
10 9
15 5
17 1

Table 6.9: Data on Temperature and Depth of Snow

Now we need to create a scatter plot and interpret the results. First we need to create a scatter plot with the temperature on the x-axis and the depth of the snow on the y-axis.

We note that there is a negative association in the graph. This means as the temperature increases, the depth of the snow decreases. We also note that it seems that the data lies in the shape of a line, making it linear.
