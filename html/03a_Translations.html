Translations

Translation is simply moving a figure from one location to another.

When we move this figure, every single point in that figure moves the same distance in the horizontal direction, and every single point in that figure moves the same distance in the vertical direction.

The figure before we move it is called the pre-image. After it is moved, we call it the image.

We can express this movement algebraically. We are moving the figure from A to A′ we read this as moving from “A to A prime.”

We express this distance of movement in the horizontal direction using x and in the vertical direction using y. Every point in the pre-image starts at a point (x,y). When we translate a figure, we move it horizontally and vertically by adding or subtracting the same amount. For example, (x,y) → (x + 4, y−3) means move every point in the pre-image to the right 4 units and down 3 units to create the image.

Note that when we do translation we use congruent figures. Congruent means that figures have the same length of their sides and the same angles as well.

Example 1

Triangle ABC has vertices of A(3, 2), B(1, 4), and C(5, 2). Graph the figure ABC′ after a translation that has the figure up 3 units and to the left 4 units.

Note that for every vertex, we added 3 to the y value and then subtracted 4 from the x value.

Example 2

Move quadrilateral ABCD → ABCD′ such that (x,y) → (x + 3, y−1)

A is at point (1, 3)
B is at point (5, 3)
C is at point (6, 1)
D is at point (2, 1)

So if (x, y) → (x + 3, y−1)
Then A (1, 3) → (1 + 3, 3−1) → A′ (4, 2)
B(5,3) → (5 + 3, 3−1) → B′(8,2)
C(6,1) → (6 + 3, 1−1) → C′(9,0)
D(2,1) → (2 + 3, 1−1) → D′(5,0)
